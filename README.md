## Projects Featured

### Seal Call Detection — CNN Audio Classifier
Convolutional neural network pipeline for classifying seal vocalisations from audio recordings. Converts raw audio to mel spectrograms and applies a CNN for binary classification. Built as final project for HDip in Computing — Data Analytics at ATU Galway.

**Stack:** Python · TensorFlow · Librosa · CNN · Spectrogram analysis
